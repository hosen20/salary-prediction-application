# Job Offers Analysis

### Project description:
Jason Miller is searching for jobs, after applying he happily received three remote job offers. The offers are with 3 different companies in the US, GB and FR. But now Jason is confused on which offer to choose.

To solve his problem he built a pipeline:

1. An online API for the ML model to predict salaries according to his profile.
2. An LLM model to compare predicted vs offered salaries and give insights about the offers, like country specific taxes, US vs FR work culture, etc ...
3. An online database called Supabase for storing, charts, images, numbers and text insights.
3. A Streamlit dashboard, summarizing the results, plotting essential charts and more.
